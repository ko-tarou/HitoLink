# 小屋のじかん - ランディングページ

雑貨・家具・小屋の専門店「小屋のじかん」のランディングページです。

## 技術スタック

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Google Fonts (Noto Sans JP, Noto Serif JP)

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ビルド

```bash
npm run build
npm start
```

## 構成

- `app/` - ページとレイアウト
- `components/` - 再利用可能なコンポーネント（Header, Hero, Concept, ProductShowcase, CoreCabin, ShopInfo, Footer）

## デザインコンセプト

「楽しむ暮らし、丁寧に暮らす。」をテーマに、Japanese Modern と Nordic Natural を融合したミニマルなデザイン。アースカラーをベースに、スクロール連動のアニメーションとパララックス効果で奥行きのある体験を提供します。
