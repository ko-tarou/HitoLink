# 森の雑貨やさん ＊MARL MARL＊

A high-end, immersive landing page for Forest General Store MARL MARL — a cozy general store in Toyama, Japan.

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Design Concept

- **Vibe:** Cozy, whimsical, sophisticated, organic, "Ghibli-esque" but modern
- **Colors:** Deep forest greens, warm beige/cream, earth tones, antique gold accents
- **Typography:** Noto Serif JP (headings), Noto Sans JP (body)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Loading Screen:** Elegant sprouting plant animation on initial load
- **Hero Section:** Full-screen parallax with floating leaves and sparkles
- **About:** Store story and concept with scroll-triggered animations
- **Products:** MARL MARL brand & general goods with hover effects
- **Info & Access:** Address, hours, and LINE contact
- **Responsive:** Mobile-first, fully responsive design
