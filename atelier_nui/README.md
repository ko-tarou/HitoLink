# Atelier NUI - ランディングページ

富山県富山市のアンティーク・ヴィンテージショップ「Atelier NUI」のランディングページです。

## 技術スタック

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Google Fonts (Cinzel, Playfair Display, Source Sans 3)

## 開発

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## ビルド

```bash
npm run build
npm start
```

## 構成

- **Header:** ロゴ・ナビゲーション・スクロール時の backdrop-blur
- **Hero:** キャッチコピー（文字ごとのフェードイン）、背景画像のパララックス
- **About:** 店舗紹介・アンティークのある暮らし
- **Collections:** カテゴリ別商品（一点もの・SOLD OUT タグ）
- **Services:** リペア・リメイクサービス
- **Access:** 住所・営業時間・Instagram・Online Shop リンク
- **Footer:** コピーライト・SNS

画像は Unsplash のプレースホルダーを使用しています。本番では差し替えてください。
