# FLOWERWORKS nico — Landing Page

A sophisticated, minimal landing page for **FLOWERWORKS nico**, a florist in Toyama, Japan with an "Antique & Natural" aesthetic.

## Tech Stack

- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**
- **Google Fonts**: Noto Serif JP (headers), Zen Kaku Gothic New (body)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Design

- **Theme:** Botanical Atelier — antique charm meets modern minimalism
- **Colors:** Cream (#FDFCF8), Sage Green, Dusty Pink, Charcoal
- **Animations:** Hero zoom-out, staggered text, scroll-triggered reveals, hover micro-interactions, smooth scroll
