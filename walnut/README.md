# 花屋ウォルナット・グローブ (Walnut Grove)

A single-page landing page for a flower shop in Toyama, Japan. Built with Next.js (App Router), TypeScript, Tailwind CSS, and Framer Motion. Design inspired by cottagecore, Studio Ghibli, and "Little House on the Prairie."

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**
- **Google Fonts:** Zen Old Mincho (headings), Zen Kaku Gothic New (body)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project Structure

- `app/layout.tsx` — Root layout, fonts, metadata
- `app/page.tsx` — Home page composition
- `app/globals.css` — Global styles and Tailwind
- `components/` — Header, Hero, About, Features, ShopInfo, Footer

## Design

- **Theme:** Cottagecore, rustic, healing, warm
- **Colors:** Sage green, warm beige, terracotta, soft white/cream
- **Animations:** Parallax hero, scroll-triggered fade-ins, hover effects, gentle float
